# ltptree
## 项目描述
本项目通过对哈工大pyltp接口进行封装，将依存关系重新组织为树状结构。

## 模型
本项目中需要用户自行下载ltp模型，解压后放入项目目录/model中。
模型地址：[百度云](http://pan.baidu.com/share/link?shareid=1988562907&uk=2738088569)
